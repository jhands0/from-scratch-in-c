# Paint

A simple paint program written in C using SDL2.

## Getting Started

### Prerequisites

Install SDL2 on your local machine

#### Debian/Ubuntu

```sh
sudo apt install libsdl2-dev
```

#### Fedora

```sh
sudo dnf install SDL2
```

#### Gentoo

```sh
sudo emerge libsdl2
```

#### From Source

```sh
git clone https://github.com/libsdl-org/SDL.git -D SDL2
cd SDL
mkdir build
../configure
make
sudo make install
```

### Installation

1. Clone the repo

```sh
git clone -b paint https://github.com/jhands0/from-scratch-in-c.git
```

2. Build the program

```sh
make release
```
